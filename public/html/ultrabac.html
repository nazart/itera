ULTRABAC

Soluciones > Backup y Data Management


Automatice los Backups de Servidores y bases de datos abiertas


UltraBac es el software de respaldo para Windows en caso de desastres, para proteger los sistemas y bases de datos de posibles pérdidas de información debido a virus, caídas de sistema, desastres naturales, errores de usuarios, hackers, etc., con el cual usted puede realizar cualquier tipo de respaldo, ya sea local o remoto, tiene la posibilidad de respaldar otros servidores y clientes que se encuentren conectados a estos servidores, también puede respaldar archivos abiertos, compartidos o en uso de aplicaciones como Exchange, Lotus Notes, SQL, Oracle, SharePoint, UNÍX, HP-UX, AIX. Solaris, Linux, etc.




Con UltraBac usted puede programar sus respaldos seleccionando el día y la hora que más le convenga realizándolos automáticamente sin necesidad de que una persona de tiempo completo, atienda estos respaldos; además se puede especificar tipo de archivos, particiones y/o discos a respaldar, los respaldos se pueden realizar a SAN, NAS, UNC Path, Disk Partition, Tivoli Storage (TSM), FTP, USB, CD, DVD. Cinta Local, Cinta Remota, Librería Local, Librería Remota, Tivoli Tape Silo.




BENEFICIOS

Es considerado el software más rápido al realizar Backup.

Protección completa y variada de los diferentes sistemas de la empresa: (Directorio activo, Exchange, SQL, Oracle Linux, mysql, Share Point, FTP, etc.).

Funciona sin inconvenientes sobre cualquier plataforma de Windows

Disaster Recovery, restaura imágenes a nivel local o de forma remota.

Las copias son de alto nivel, no se distorsiona la data respaldada.

Trabaja perfectamente con diferentes sistemas de almacenamiento
